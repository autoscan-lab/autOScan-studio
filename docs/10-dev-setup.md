# Development Setup

## Prerequisites

1. macOS latest stable + Xcode
2. Access to private `autOScan-engine`
3. SSH access to university servers (for integration testing)

## Initial Steps

1. Create the macOS app project in `apps/macos/`
2. Implement shell layout and local file browsing
3. Add engine adapter with mock responses first
4. Replace mocks with real engine integration

## Definition of Done (first runnable)

1. Open folder
2. Select file from tree and view in editor
3. Trigger one engine action and display result
