# v1 Scope (macOS)

## In Scope

1. Workspace/folder open
2. Left file tree
3. Center code editor/viewer
4. Right autOScan tool panel
5. Bottom terminal panel
6. SSH quick connect profiles
7. Run core grading actions via `autOScan-engine`

## Out of Scope

1. Git features (commit/branch UI)
2. Full LSP parity with VS Code/Zed
3. Real-time multi-user collaboration
4. Cloud sync/auth

## Release Philosophy

Ship a stable, useful grading workbench first. Expand after real TA usage feedback.
